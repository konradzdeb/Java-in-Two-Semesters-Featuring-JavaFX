当你向plot()提供一系列数字时，它假设第一个数据点对应的X坐标值为0，但我们的第一个 点对应的x值为1。为改变这种默认行为，我们可以给plot()同时提供输入值和输出值：

mpl_squares.py

import matplotlib.pyplot as plt

input_values = [1, 2, 3, 4, 5] squares = [1, 4, 9, 16, 25] plt.plot(input_values, squares, linewidth=5)
