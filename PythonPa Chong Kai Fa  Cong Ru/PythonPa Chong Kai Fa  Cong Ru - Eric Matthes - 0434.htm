下面来确保用于查看信息的页面的样式也是合适的，首先来设置topics页面的样式：

{% for topic in topics %}

</li>

{% empty %}

{% endfor %}

</ul>

❸   <h3><a href="{% url 'learning_logs:new_topic' %}">Add new topic</h3>

我们不需要标签{% load bootstraps %}，因为我们在这个文件中没有使用任何bootstrap3自 定义标签。我们在header块中添加了标题Topics （见❶）。为设置每个主题的样式，我们将它们都 设置为<h3>元素，让它们在页面上显得大些（见❷）；对于添加新主题的链接，也做了同样的处 理（见❸）。
