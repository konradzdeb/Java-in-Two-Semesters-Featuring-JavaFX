安装Python


Python有多个不同的版本，而在各种操作系统中安装它的方式有 很多。如果第1章介绍的方法不管用，或者你要安装非系统自带的 Python版本，这个附录可提供帮助。

A.1 在Linux系统中安装Python

几乎所有Linux系统都默认安装了Python,但你可能想使用非默认版本。如果是这样，请首 先确定已安装的Python版本。

A.1.1 确定已安装的版本

打开一个终端窗口，并执行如下命令：

$ python --version

Python 2.7.6

输出表明默认版本是2.7.6,但系统可能还安装了一个Python 3版本。为核实这一点，请执行 如下命令：

$ python3 --version

Python 3.5.0

输出表明也安装了Python 3.5.0。安装新版本前，有必要执行上述两个命令。
